Visibility Calculation

This calculates various visibility measures relating to your data. The measures are Total Visibility, Visibility Variation and Visibility Vector Resultant.

Options

  • Window Size - This is an odd number specifying the size of window to pass over your data. The visibility is calculated for each window.
  • Viewing Height - This is the height of the observer above the surface.

References

Cooper,G.R.J. 2005. Analysing potential field data using visibility. Computers and Geosciences, 31, p.877-881
